# AI-Powered Student Performance Prediction & Early Warning System

## Problem Statement

Teachers often identify weak students only after final examinations. This project predicts students who are at academic risk before final exams and provides personalized recommendations for early intervention.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Matplotlib
- Seaborn

## Machine Learning Algorithm

- Random Forest Classifier

## Features

- Student Risk Prediction
- Early Warning System
- Personalized Recommendations
- Feature Importance Analysis
- Interactive Streamlit Dashboard

## Future Scope

- School ERP Integration
- Parent Email Notifications
- Real-Time Dashboard
- Multi-School Analytics
